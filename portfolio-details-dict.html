App de dicionário com diferentes tipos de contas

O aplicativo permite ao usuário realizar consultas na API OpenDictionary, retornando informações sobre a palavra pesquisada, além de efetivamente utilizar cache para reutilização de palavras já pesquisadas. O app também tem sistema de contas trial e premium, com diferentes limitações de uso.
